# Ferramentas Utilizadas

Este documento apresenta as ferramentas utilizadas na criação do podcast **Dados em Ação**, explicando o papel de cada uma dentro da esteira de produção do projeto.

---

## 1. ChatGPT

### Finalidade

O ChatGPT foi utilizado como ferramenta de apoio para:

- Definir o tema do podcast;
- Criar sugestões de nome e subtítulo;
- Estruturar os prompts do projeto;
- Gerar o roteiro do episódio;
- Revisar a linguagem;
- Apoiar a documentação do repositório.

### Uso no Projeto

A ferramenta foi utilizada principalmente nas etapas de ideação, organização textual e documentação.

O objetivo não foi apenas gerar conteúdo automaticamente, mas construir uma base estruturada, revisar o posicionamento do podcast e garantir que o projeto estivesse alinhado ao público de IA, análise de dados e BI.

---

## 2. ElevenLabs

### Finalidade

O ElevenLabs foi utilizado para gerar a narração do episódio a partir do roteiro final.

### Uso no Projeto

A ferramenta converteu o texto do roteiro em uma narração com voz sintética, buscando um tom:

- Claro;
- Profissional;
- Natural;
- Acessível;
- Adequado para podcast educativo.

O áudio gerado serviu como base para a edição final do episódio.

---

## 3. Canva

### Finalidade

O Canva foi utilizado para criar a capa visual do podcast.

### Uso no Projeto

A capa foi criada com foco em uma identidade visual relacionada a:

- Inteligência Artificial;
- Análise de dados;
- Tecnologia;
- Dashboards;
- Tomada de decisão;
- Pensamento crítico.

O objetivo da capa foi criar uma apresentação visual moderna, limpa e profissional para o podcast **Dados em Ação**.

---

## 4. CapCut

### Finalidade

O CapCut foi utilizado para editar e finalizar o áudio do podcast.

### Uso no Projeto

A ferramenta foi utilizada para:

- Importar a narração;
- Ajustar o volume da voz;
- Adicionar trilha sonora de fundo;
- Realizar cortes simples;
- Aplicar fade out no final;
- Exportar o áudio final em MP3.

A edição teve como objetivo manter o áudio limpo, agradável e com foco na clareza da narração.

---

## 5. Git

### Finalidade

O Git foi utilizado para controle de versão do projeto.

### Uso no Projeto

Com o Git, foi possível registrar as etapas de criação dos arquivos, organizar os commits e manter um histórico de evolução do projeto.

---

## 6. GitHub

### Finalidade

O GitHub foi utilizado para hospedar o repositório do projeto e entregar o desafio.

### Uso no Projeto

O repositório reúne:

- README do projeto;
- Prompts utilizados;
- Roteiro do episódio;
- Capa do podcast;
- Áudio final;
- Documentação das ferramentas utilizadas.

---

## Resumo da Esteira de Produção

```text
Ideia e posicionamento
        ↓
Criação dos prompts
        ↓
Geração do roteiro
        ↓
Criação da capa
        ↓
Geração da narração
        ↓
Edição do áudio
        ↓
Organização no GitHub
        ↓
Entrega do projeto
```

---

## Observação Final

As ferramentas de Inteligência Artificial foram utilizadas como apoio ao processo criativo e produtivo.

A curadoria, a escolha do tema, a revisão da mensagem, a organização do projeto e a validação final permaneceram como responsabilidades humanas.
